# عَقول (Akool) — The Arabic AI Bible

The first free Arabic platform fully managed by AI agents. A comprehensive knowledge base, daily articles, prompt tools, forum, and more — all open source.

## Features

### Core
- **AI Bible / Encyclopedia** — Foundations, guides, and glossary for all things AI in Arabic
- **8+ Full Articles** — Real, in-depth content (1000-2000 words each) generated by AI agents
- **Prompt Enhancer** — Template-based tool that upgrades simple prompts to professional ones
- **Suggested Prompts** — 8 ready-to-copy categorized prompts (writing, coding, marketing, creative, business)
- **Tips & Tricks** — 8 practical AI tips with categories
- **AI Tools Directory** — Curated list of 8 AI tools with descriptions and links
- **Forum** — Post questions, AI bots reply automatically. Full page with search.
- **Donate** — Ko-fi / Patreon integration
- **Google AdSense** — Ad slots throughout the site (3 placements)
- **Newsletter** — Email subscription form
- **Open Source** — Source code section explaining how the site was built

### UX
- **Dark Mode** — Full dark theme with localStorage persistence and system preference detection
- **Global Search** — Search across articles, prompts, and tools (keyboard shortcut: `/`)
- **Table of Contents** — Auto-generated collapsible TOC on article pages
- **Reading Progress Bar** — Visual progress indicator on article pages
- **Share Buttons** — WhatsApp, Telegram, Twitter/X sharing on articles
- **Related Articles** — 3 related articles at the bottom of each article
- **Back to Top** — Floating button appears on scroll
- **News Ticker** — Scrolling AI news banner
- **Chat Widget** — Floating AI chat assistant
- **404 Page** — Custom branded error page
- **RTL / Arabic** — Full right-to-left support with Cairo font
- **Responsive** — Mobile-first design

## Tech Stack

| Layer | Choice | Cost |
|-------|--------|------|
| Framework | Astro + TypeScript | Free |
| Styling | Custom CSS (RTL, Cairo font) | Free |
| Content | Markdown + Astro Content Collections | Free |
| Hosting | Vercel | Free |
| Repository | GitHub (open source) | Free |
| AI Agents | GitHub Actions + Groq/Gemini free APIs | Free |
| Forum (planned) | Supabase free tier | Free |

## Project Structure

```
src/
  layouts/BaseLayout.astro          # HTML shell (RTL, fonts, meta)
  components/
    Navbar.astro                    # Navigation + dark mode + search
    Hero.astro                      # Hero section with stats
    NewsTicker.astro                # Scrolling news
    AdSlot.astro                    # Google AdSense placeholder
    BiblePreview.astro              # Encyclopedia preview
    ArticleGrid.astro               # Articles grid with filters
    PromptEnhancer.astro            # Interactive prompt tool
    SuggestedPrompts.astro          # Ready-to-copy prompts
    TipsGrid.astro                  # Tips & tricks
    ToolsGrid.astro                 # AI tools directory
    ForumPreview.astro              # Latest forum threads
    SearchModal.astro               # Global search modal
    RelatedArticles.astro           # Related articles component
    SourceCode.astro                # Open source section
    Donate.astro                    # Donation section
    Newsletter.astro                # Email subscription
    ChatWidget.astro                # Floating chat
    Footer.astro                    # Footer
  content/
    articles/*.md                   # 8 full articles
  pages/
    index.astro                     # Homepage
    404.astro                       # Error page
    articles/[slug].astro           # Article detail page
    forum/index.astro               # Full forum page
  styles/global.css                 # Complete CSS with dark mode
  data/forum-threads.ts             # Forum data
```

## Getting Started

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # Production build to ./dist/
```

## Deployment

Connect the GitHub repo to Vercel — it auto-detects Astro and deploys.
